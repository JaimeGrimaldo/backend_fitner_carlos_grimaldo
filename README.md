# BACK-END

Repositorio que almacena el Backend de **Fitner**

## Alumno

Carlos Mario Grimaldo Moreno

## Tecnología usada

- NodeJs
- MySQL
- Express

## ¿Como iniciar?

1. Primeramente se deben de instalar las dependencias

```bash
npm install
```

2. Comprobar que el Front-end y Back-end no ocupen los mismos puertos, en este caso el back-end corre en el puerto **3000**
3. Arrancar servidor con el comando:

```bash
node index.js
```
